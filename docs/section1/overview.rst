Introduction to Jetstream2 and Data
================================

In this unit we introduce the computational resources and software environments that will be used
throughout the workshop. Concepts covered will include the Linux command line interface, python, Jetstream2 (JS2), Jupyter notebooks, basic Python, and exploratory data
analysis. Mastery of these concepts is essential for successful completion of the hands-on exercises
to come.
